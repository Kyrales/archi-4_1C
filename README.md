# Архитектура С4 на 1С

## Цель
Формирование архитектурных документов принятия решений (ADR), с построением диаграмм на С4 из 1С:
- Возможность фиксация ADR, с помощью документа "Архитектурные решение"
- Возможность ведения реестра точек интеграции
- Построение всех диаграмм с использованием PlantUML (рендеринг кодом) на основе библиотеки: https://github.com/plantuml-stdlib/C4-PlantUML
- Формирование диаграммы Функций потоков данных бизнес-доменов на основе документа "Архитектурное решение" 
- Формирование диаграммы Архитектура программных систем и интеграций на основе документа "Архитектурное решение"
- Формирование отчета по реестру точек интеграций

## Поставка
Подсистема поставляется в виде расширения, которое может быть встроенно в конфигурации на базе "Библиотека Стандартных подсистем" редакции 3.1

## Установка и настройка

По-умолчанию расширение работает с демо базой "Библиотеки Стандартных подсистем" версии 3.1.11. 
1. Скачать и установить последнюю версию расширения IntegrationPoint_*.cfe в конфигурацию на базе "Библиотека Стандартных подсистем" редакции 3.1
2. В расширении наследовать справочник Организации и в определяемом типе Организация, указать новый тип справочника
